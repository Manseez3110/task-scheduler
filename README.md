# Task Scheduler

A React application for scheduling and managing your tasks efficiently.

## Features ✨
- ✅ Create new tasks
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Set due dates and times
- ✅ Assign priority levels (High, Medium, Low)
- ✅ Add tags/categories to tasks
- ✅ Automatic local storage persistence

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Manseez3110/task-scheduler.git
   cd task-scheduler
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
   The application will open at `http://localhost:3000`

## Usage
- **Add Task**: Fill in the form and click "Add Task"
- **Edit Task**: Click the "Edit" button on any task
- **Complete Task**: Check the checkbox next to a task
- **Delete Task**: Click the "Delete" button
- **Set Priority**: Choose High, Medium, or Low priority
- **Add Tags**: Separate multiple tags with commas
- **Due Dates**: Set specific dates and times for your tasks

All tasks are automatically saved to your browser's local storage.

## Technologies Used
- React 18
- CSS3
- Local Storage API
